#### [172. 阶乘后的零](https://leetcode-cn.com/problems/factorial-trailing-zeroes/)

难度简单280

给定一个整数 *n*，返回 *n*! 结果尾数中零的数量。

**示例 1:**

```
输入: 3
输出: 0
解释: 3! = 6, 尾数中没有零。
```

**示例 2:**

```
输入: 5
输出: 1
解释: 5! = 120, 尾数中有 1 个零.
```

**说明:** 你算法的时间复杂度应为 *O*(log *n*) 。





## 数有多少个5就行，但是超出时间限制了

```c++
class Solution {
public:
    int trailingZeroes(int n) {
        int res = 0;
        for(int i = n; i > 0; --i) {
            for(int j = i; j > 0 && j % 5 == 0; j /= 5) {
                res++;
            }
        }
        return res;
    }
};
```

class Solution {
public:
    int trailingZeroes(int n) {
        int res = 0;
        while(n > 0) {
            res += n / 5;
            n /= 5;
        }
        return res;
    }
};
